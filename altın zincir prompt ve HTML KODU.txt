# 🏗️ BÖLÜM 1: ANDROID TEKNİK MİMARİSİ (JAVA)

### 1. `AndroidManifest.xml`
*   İnternet izni: `<uses-permission android:name="android.permission.INTERNET" />`
*   Ağ trafiği izni: `android:usesCleartextTraffic="true"`

### 2. `MainActivity.java` (WebView & Köprü)
*   **WebView Ayarları (ZORUNLU):**
    *   `setJavaScriptEnabled(true)`
    *   `setDomStorageEnabled(true)` (LocalStorage çalışmalı)
    *   `setAllowUniversalAccessFromFileURLs(true)` (CORS ve Scraping engellerini kaldırmak için şart)
    *   `setAllowFileAccessFromFileURLs(true)`
*   **JS Interface (`AndroidBridge`):** HTML ile Android'in konuşması için şu metodları ekle:
    *   `triggerNotification(String title, String message)`: JS tarafındaki alarm tetiklenince Android native bildirimi (NotificationChannel) gönder.
    *   `purchaseProSubscription()`: Google Play Billing akışını başlat (Simülasyon logu yeterli).
    *   `saveWidgetData(String json)`: Favori listesi güncellendiğinde veriyi SharedPreferences'a kaydet (İleride Widget ekleneceği için altyapı hazır olsun).

# 🧠 BÖLÜM 2: JAVASCRIPT LOGIC (VERİ VE İŞLEYİŞ)

Aşağıdaki referans HTML kodunun `<script>` kısmını şu mantıkla **baştan aşağıya eksiksiz** yaz:

### A. CANLI VERİ KAYNAKLARI (PİYASALAR & ÇEVİRİCİ)
*   **Altın & Döviz:** `https://finans.truncgil.com/today.json` (Fetch).
*   **Kripto:** `https://api.binance.com/api/v3/ticker/price` (Fetch).
*   **Hisse Senetleri (BIST):** `https://query1.finance.yahoo.com/v8/finance/chart/{SYMBOL}.IS?interval=1d&range=1d` (Fetch - Son kapanış fiyatını al).
    *   *Semboller:* THYAO, ASELS, GARAN, EREGL.

### B. DETAY EKRANI VE GRAFİK (CHART.JS)
Kullanıcı bir varlığa tıkladığında açılan detay ekranı:
1.  **Tarihsel Veri:** Yahoo Finance API kullanarak grafiği çiz.
    *   Döviz/Altın: `TRY=X`, `XAUTRY=X`.
    *   Kripto: `BTC-USD`.
    *   Hisse: `THYAO.IS`.
2.  **Zaman Butonları:** (GÜN, HAFTA, AY, YIL) butonlarına basınca API'ye `interval` ve `range` parametrelerini değiştirerek istek at ve grafiği güncelle.
3.  **Dinamik İstatistikler:** Grafiğin altındaki "En Yüksek", "En Düşük", "Açılış", "Kapanış" kutularını, API'den gelen o anki veri setinin matematiğini yaparak (Min/Max bularak) doldur. Statik veri olmasın.

### C. HABERLER VE TAKVİM
1.  **Haberler:** `https://tr.investing.com/rss/news.rss` adresini çek, XML'i parse et ve "Haberler" sekmesine kart olarak bas.
2.  **Takvim:** `Investing.com/economic-calendar` adresine fetch at (Android izni sayesinde çalışır), HTML'i parse et ve Türkiye/ABD verilerini tabloya bas.

### D. FAVORİLER VE ALARM SİSTEMİ
1.  **Favori Mantığı:** Yıldız ikonuna basınca `LocalStorage`'a kaydet. "Favorilerim" sekmesinde sadece bunlar görünsün.
2.  **Alarm (Zil) Mantığı:** Favoriler sekmesinde varlığın yanında **Zil** ikonu vardır.
    *   Tıklanınca Sarı (Aktif) olur. Durumu LocalStorage'da sakla.
    *   **OTOMATİK KONTROL:** `setInterval` ile veriler her güncellendiğinde (örn: 10sn), eğer zil açık olan bir varlık bir önceki fiyata göre **%0.5 (Binde Beş)** ve üzeri değişim yaptıysa `AndroidBridge.triggerNotification` metodunu çağır.

### E. PRO MOD (KISITLAMALAR)
1.  **Sınırlar:** Varsayılan olarak kullanıcı Free moddadır.
    *   En fazla **5 Favori** eklenebilir.
    *   En fazla **3 Alarm** kurulabilir.
2.  **Engel:** Kullanıcı bu limitleri aşmaya çalışırsa "Pro'ya Geçin" uyarısı ver veya Pro Modal'ı aç.
3.  **Satın Alma:** Pro Modal'daki butona basılınca Android tarafına sinyal gönder. Başarılı olursa sınırları kaldır.

### F. AYARLAR (GÖRÜNÜM)
1.  **Punto Slider:** Sağa/sola çekildiğinde HTML'in `root font-size` değerini (px cinsinden) değiştir. CSS `rem` kullandığı için tüm uygulama (ikonlar, yazılar, kartlar) anlık büyüyüp küçülmeli.
2.  **Tema:** Dark/Light mod seçimi LocalStorage'da saklanmalı.

Mevcut WebView tabanlı finans uygulamama, Android ana ekranında çalışacak bir **Widget (Araç Takımı)** eklemek istiyorum.

**DURUM:** Uygulama şu an `LocalStorage` kullanıyor ve `MainActivity` içinde bir `AndroidBridge` (JavascriptInterface) kurulu. HTML tarafından `saveWidgetData(String json)` metodu ile favori listesi JSON formatında Android tarafına gönderiliyor.

# 1. WIDGET TASARIM KURALLARI (UI)
*   **Boyut:** 4x3 hücre (Resizable).
*   **Arka Plan:** Koyu Mod (`#121212`), Köşeler yuvarlak (16dp veya 22dp).
*   **Header (Üst Kısım):**
    *   Sol Üst: Uygulama Logosu ve "AltınZincir" yazısı (Altın Sarısı #FFD700).
    *   **ETKİLEŞİM:** Header alanına (Logo veya Yazı) tıklandığında **Uygulama Açılmalı (`MainActivity`)**.
*   **Liste (Gövde):**
    *   `ListView` kullanılacak.
    *   Her satırda: Sol tarafta Varlık İkonu ve Kodu (USD, GRAM vb.), Sağ tarafta Fiyat ve Değişim Oranı.
    *   **ÖNEMLİ:** Satırlarda "Zil/Bildirim" ikonu **OLMAYACAK**. Sadece veri gösterilecek.

# 2. TEKNİK MİMARİ & VERİ AKIŞI
Veri akışını şu şekilde kurgula:

1.  **Veri Kaydı (`MainActivity.java`):**
    *   JS Interface içindeki `saveWidgetData` metodu çalıştığında, gelen JSON verisini `SharedPreferences` içine "widget_favorites_json" anahtarıyla kaydet.
    *   Kaydettikten sonra `AppWidgetManager` kullanarak widget'a "Veri değişti, güncellen" sinyali gönder (`notifyAppWidgetViewDataChanged`).

2.  **Veri Okuma (`WidgetService.java`):**
    *   Widget'ın `RemoteViewsFactory` servisi, `SharedPreferences`'tan bu JSON verisini okumalı.
    *   JSON'u parse edip listeyi doldurmalı.

public class FavoritesWidget extends AppWidgetProvider {

    static void updateAppWidget(Context context, AppWidgetManager appWidgetManager, int appWidgetId) {
        // Layout'u bağla
        RemoteViews views = new RemoteViews(context.getPackageName(), R.layout.widget_layout);

        // 1. HEADER TIKLAMA (Uygulamayı Aç)
        Intent intentApp = new Intent(context, MainActivity.class);
        PendingIntent pendingIntent = PendingIntent.getActivity(context, 0, intentApp, PendingIntent.FLAG_UPDATE_CURRENT | PendingIntent.FLAG_IMMUTABLE);
        views.setOnClickPendingIntent(R.id.widget_header, pendingIntent);

        // 2. LİSTEYİ DOLDUR (Service Bağlantısı)
        Intent intentService = new Intent(context, WidgetService.class);
        views.setRemoteAdapter(R.id.widget_list_view, intentService);
        views.setEmptyView(R.id.widget_list_view, R.id.empty_view);

        appWidgetManager.updateAppWidget(appWidgetId, views);
    }

    @Override
    public void onUpdate(Context context, AppWidgetManager appWidgetManager, int[] appWidgetIds) {
        for (int appWidgetId : appWidgetIds) {
            updateAppWidget(context, appWidgetManager, appWidgetId);
        }
    }
}

public class WidgetService extends RemoteViewsService {
    @Override
    public RemoteViewsFactory onGetViewFactory(Intent intent) {
        return new ListRemoteViewsFactory(this.getApplicationContext());
    }
}

class ListRemoteViewsFactory implements RemoteViewsService.RemoteViewsFactory {
    private Context context;
    private List<JSONObject> dataList = new ArrayList<>();

    public ListRemoteViewsFactory(Context context) { this.context = context; }

    @Override
    public void onDataSetChanged() {
        dataList.clear();
        // SharedPreferences'tan "widget_favorites_json" anahtarını oku
        SharedPreferences prefs = context.getSharedPreferences("AltinZincirPrefs", Context.MODE_PRIVATE);
        String jsonStr = prefs.getString("widget_favorites_json", "[]");
        
        try {
            JSONArray jsonArray = new JSONArray(jsonStr);
            for (int i = 0; i < jsonArray.length(); i++) {
                dataList.add(jsonArray.getJSONObject(i));
            }
        } catch (JSONException e) { e.printStackTrace(); }
    }

    @Override
    public RemoteViews getViewAt(int position) {
        if (position >= dataList.size()) return null;
        
        RemoteViews views = new RemoteViews(context.getPackageName(), R.layout.widget_item);
        JSONObject item = dataList.get(position);

        try {
            views.setTextViewText(R.id.item_code, item.getString("code"));
            views.setTextViewText(R.id.item_name, item.getString("name"));
            views.setTextViewText(R.id.item_price, "₺" + item.getString("price"));
            
            // Renk ayarı (Örn: değişim pozitifse yeşil)
            // views.setTextColor(R.id.item_change, Color.GREEN); 
        } catch (JSONException e) { e.printStackTrace(); }

        return views;
    }

    // Diğer zorunlu metodlar (onCreate, getCount vb.) standart...
    @Override public void onCreate() {}
    @Override public int getCount() { return dataList.size(); }
    @Override public long getItemId(int position) { return position; }
    @Override public RemoteViews getLoadingView() { return null; }
    @Override public int getViewTypeCount() { return 1; }
    @Override public boolean hasStableIds() { return true; }
    @Override public void onDestroy() {}
}

        async function fetchLiveMarket() {
            try {
                // 1. Kripto (Binance)
                fetch('https://api.binance.com/api/v3/ticker/price')
                    .then(res => res.json())
                    .then(data => {
                        data.forEach(coin => {
                            if(coin.symbol === 'BTCUSDT') updateAssetRate('BTC', parseFloat(coin.price) * 32); // Mock USD multiplier
                            if(coin.symbol === 'ETHUSDT') updateAssetRate('ETH', parseFloat(coin.price) * 32);
                        });
                    }).catch(()=>console.log("Binance Fail"));

                // 2. Altın & Döviz (Truncgil) - Eğer çalışmazsa GenelPara'ya düşer
                const res = await fetch('https://finans.truncgil.com/today.json').catch(e => null);
                if(res && res.ok) {
                    const data = await res.json();
                    if(data.USD) updateAssetRate('USD', parseFloat(data.USD.satis));
                    if(data.EUR) updateAssetRate('EUR', parseFloat(data.EUR.satis));
                    if(data['Gram Altın']) updateAssetRate('GRAM', parseFloat(data['Gram Altın'].satis));
                } else {
                    throw new Error("Truncgil Down");
                }
            } catch (e) {
                // FALLBACK: GenelPara
                console.log("Fallback to GenelPara...");
                try {
                    const resGP = await fetch('https://api.genelpara.com/embed/altin.json');
                    const dataGP = await resGP.json();
                    if(dataGP.GA) updateAssetRate('GRAM', parseFloat(dataGP.GA.satis.replace(',','.')));
                } catch(err) {
                     // Hiçbiri çalışmazsa eski değerler kalır (Simülasyon yok)
                }
            }
            updateUI();
        }

        function updateAssetRate(code, rate) {
            const asset = assetsMap.find(a => a.code === code);
            if(asset) asset.rate = rate;
        }

        function getFormattedPrice(rateInTRY) {
            let parity = 1;
            let symbol = '₺';
            let fraction = 2;

            if (defaultCurrency === 'USD') {
                const usd = assetsMap.find(a => a.id === 'usd');
                parity = usd ? usd.rate : 32.0; 
                symbol = '$';
            } else if (defaultCurrency === 'EUR') {
                const eur = assetsMap.find(a => a.id === 'eur');
                parity = eur ? eur.rate : 35.0;
                symbol = '€';
            } else if (defaultCurrency === 'GBP') {
                const gbp = assetsMap.find(a => a.id === 'gbp');
                parity = gbp ? gbp.rate : 40.0;
                symbol = '£';
            }

            const convertedValue = rateInTRY / parity;
            if (convertedValue < 1) fraction = 4;
            if (convertedValue < 0.01) fraction = 6;
            return symbol + convertedValue.toLocaleString('tr-TR', {minimumFractionDigits: fraction, maximumFractionDigits: fraction});
        }

        async function fetchLiveMarket() {
            try {
                // FALLBACK SİSTEMİ: TRUNCGIL -> GENELPARA
                const res = await fetch('https://finans.truncgil.com/today.json').catch(e => null);
                if(res && res.ok) {
                    const data = await res.json();
                    if(data.USD) updateAssetRate('USD', parseFloat(data.USD.satis));
                    if(data.EUR) updateAssetRate('EUR', parseFloat(data.EUR.satis));
                    if(data['Gram Altın']) updateAssetRate('GRAM', parseFloat(data['Gram Altın'].satis));
                } else {
                    // Fallback
                    const resGP = await fetch('https://api.genelpara.com/embed/altin.json');
                    const dataGP = await resGP.json();
                    if(dataGP.GA) updateAssetRate('GRAM', parseFloat(dataGP.GA.satis.replace(',','.')));
                }
                updateUI();
            } catch (e) {}
        }

        function updateChart(period, btn, color) {
            if(btn) { document.querySelectorAll('.t-tab').forEach(b => b.classList.remove('active')); btn.classList.add('active'); }
            const ctx = document.getElementById('detailChart').getContext('2d');
            if(chartInstance) chartInstance.destroy();
            let labels = []; let count = 10; let volatility = 0.02;
            if(period === '1D') { labels = ['09:00','11:00','13:00','15:00','17:00','19:00']; count=6; volatility=0.01; }
            else if(period === '1W') { labels = ['Pzt','Sal','Çar','Per','Cum','Cmt','Paz']; count=7; volatility=0.03; }
            else if(period === '1M') { labels = ['H1','H2','H3','H4']; count=4; volatility=0.08; }
            else if(period === '1Y') { labels = ['Oca','Mar','May','Tem','Eyl','Kas']; count=6; volatility=0.20; }
            const data = [];
            for(let i=0; i<count; i++) { let randomFactor = 1 + ((Math.random() - 0.5) * volatility); data.push(currentDetailPrice * randomFactor); }
            data[data.length - 1] = currentDetailPrice;
            document.getElementById('stat-high').innerText = getFormattedPrice(Math.max(...data));
            document.getElementById('stat-low').innerText = getFormattedPrice(Math.min(...data));
            document.getElementById('stat-open').innerText = getFormattedPrice(data[0]);
            document.getElementById('stat-close').innerText = getFormattedPrice(data[data.length - 1]);
            const drawColor = '#FFD700'; 
            const gradient = ctx.createLinearGradient(0,0,0,300);
            gradient.addColorStop(0, 'rgba(255, 215, 0, 0.5)'); 
            gradient.addColorStop(1, 'transparent');
            chartInstance = new Chart(ctx, { type: 'line', data: { labels: labels, datasets: [{ data: data, borderColor: drawColor, backgroundColor: gradient, fill: true, tension: 0.4, pointRadius: 3 }] }, options: { responsive: true, maintainAspectRatio: false, plugins: { legend: {display:false} }, scales: { x: { grid: {display:false}, ticks: {color: '#6B7280'} }, y: { display: false } } } });
        }
